# P5 — Calculadora ROI + Otimização de Budget

Calculadora de ROI/ROAS com otimização linear de alocação de budget entre campanhas.

## Stack
- **Frontend:** Next.js (Cloudflare Pages)
- **API:** Cloudflare Workers (Hono + TypeScript)
- **Otimização:** Algoritmo proporcional ao ROAS
- **Banco:** Cloudflare D1 (SQLite)
- **Cron:** Cloudflare Cron Triggers
- **Deploy:** Cloudflare Pages + Workers
